# FileDownloadService

Triggers browser file downloads for temporary files generated by the backend by setting location.href to the download endpoint URL.

## Public interface

- `downloadTempFile(file: FileDto): void`

## Dependencies

- [[app-consts]] provides the remote service base URL for constructing the download endpoint

## Used by

- [[utils-module]]

## External dependencies

- `@angular/core`

## Notes

Uses location.href assignment rather than an anchor click — the code comment acknowledges this causes a page reload in Firefox.
