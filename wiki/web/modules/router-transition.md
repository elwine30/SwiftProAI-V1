# RouterTransitionAnimations

Provides Angular animation trigger factory functions for route transition effects used in app and account modules.

## Public interface

- `appModuleAnimation(): AnimationTriggerMetadata`
- `accountModuleAnimation(): AnimationTriggerMetadata`
- `slideFromBottom(): AnimationTriggerMetadata`
- `slideFromUp(): AnimationTriggerMetadata`
